# 🚀 My First Web on Fedora (Surface Go 1)

โปรเจกต์เว็บแรกที่สร้างขึ้นบนระบบปฏิบัติการ **Fedora Linux** โดยใช้เครื่อง **Surface Go 1** ## 🛠️ Technologies Used
- **HTML5**: โครงสร้างเว็บไซต์
- **CSS3**: ตกแต่งหน้าตา (Dark Mode & Red Theme)
- **JavaScript**: เพิ่มลูกเล่น Magic Button เปลี่ยนสีเส้นขอบ
- **Git & GitHub**: สำหรับ Version Control และการ Deploy

## 📸 Features
- หน้าเว็บรองรับ Dark Mode ถนอมสายตา
- ปุ่ม "Magic Button" สำหรับสุ่มสีเส้นขอบแบบ Real-time
- พัฒนาบนสภาพแวดล้อม Linux เต็มตัว

## 🐧 My Fedora Dev Setup
- **Editor**: VS Code
- **Runtime**: Node.js
- **Browser**: Google Chrome (สำหรับทดสอบ)

## 📖 How to run
1. Clone โปรเจกต์นี้ลงเครื่อง
2. เปิดไฟล์ `index.html` ด้วย Browser หรือใช้ Live Server ใน VS Code
